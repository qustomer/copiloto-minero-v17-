# Copiloto Minero v17

Terminal pericial digital para auditoría socioambiental, pricing consultivo y generación automática de propuestas técnicas y comerciales.

## Funcionalidades principales

1. Ingesta híbrida de evidencia documental (PDF + metadata)
2. Validación pericial de evidencia (ICG + EDDF anti-eco)
3. Scoring Heptagonal ponderado por territorio y mineral
4. Análisis de fricción socioambiental
5. Modelo de supervivencia del proyecto (Cox)
6. War Room con simulador de soluciones
7. Pricing automático basado en riesgo
8. Generación de reportes PDF duales (Comercial + Técnico)
